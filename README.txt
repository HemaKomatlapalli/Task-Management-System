**How to run the application**
1. Open your XAMPP Control Panel and start Apache and MySQL.
2. Extract the source code zip file.
3. Copy the extracted source code folder and paste it into the XAMPP's "htdocs" directory.
4. Browse the PHPMyAdmin in a browser. i.e. http://localhost/phpmyadmin
5. Create a new database named trs_db.
6. Import the provided SQL file. (trs_db.sql) 
7. Browse the Task Reminder System in a browser. i.e. http://localhost/php-trs/.
6. Use the sample logins below to access the admin and user accounts
